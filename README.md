# French_Radio_Pack

Playlist de radios françaises pour **Euro Truck Simulator 2 et American Truck Simulator**.

Ce projet propose une liste de flux radio français fonctionnels et régulièrement mis à jour pour améliorer votre immersion sur les routes d’Europe.

## 📻 Contenu:

- 📻 78 radios françaises optimisées (version bêta)
  
- 🔊 Flux stables et remis à neuf et tester

- ⚙️ Compatible ETS2/ATS (Toutes Versions du jeu)

## 🎯 Objectif du projet:

Créer un pack radio europeanne propre, stable et mis à jour régulièrement pour la communauté ETS2.

❤️ Projet 100% gratuit

## 📦 Installation

1. Téléchargez le fichier `live_streams.sii`
2. Copiez-le dans le dossier : `Documents\Euro Truck Simulator 2\` ou `Documents\American Truck Simulator\`
